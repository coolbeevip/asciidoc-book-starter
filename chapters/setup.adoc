== 环境准备

在本地渲染图书，需要 Ruby 以及 Asciidoctor 工具链。本仓库的 Makefile 默认你已经安装了 `asciidoctor` 与 `asciidoctor-pdf`。

. 安装依赖（若尚未安装）：
+
[source,bash]
----
gem install asciidoctor asciidoctor-pdf
----

. 构建 HTML 版本：
+
[source,bash]
----
make html
----

. 构建 PDF 版本：
+
[source,bash]
----
make pdf
----

如果你偏好其他工具链，可调整 Makefile 或编写自己的构建脚本以适配环境。
